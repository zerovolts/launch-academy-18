Document Ready

One of the tricky aspects of JavaScript, especially when manipulating the DOM, is making sure the page has loaded before executing JavaScript.

Your task is to create a function called ready() which will return a Promise. This Promise will resolve when the document.readyState has changed from loading to some other state. The different states of document.readyState are:

  • loading
  • interactive
  • complete

You will see the text below change from "Document is loading...", to "Document has loaded!" after you have implemented the ready() Promise correctly.

Hint: an event listener on readystatechange could be helpful, here.
