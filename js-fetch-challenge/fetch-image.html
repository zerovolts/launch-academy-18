Fetch Image

Write JavaScript to fetch the image located at /images/logo.png and insert the image into the DOM.

fetch will return a response object. If we fetch the image at the above location, we then need to use the response object to create a new image element. 

      let imageBlob = response.blob()
      let objectURL = URL.createObjectURL(imageBlob)  // https://developer.mozilla.org/en-US/docs/Web/API/URL/createObjectURL

      let imageElement = document.createElement('img')
      imageElement.src = objectURL

Now, we can simply append the imageElement to the DOM.
